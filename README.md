## 📂 Main Projects

### 1. LearnIT (E-Learning Platform)
- **Role**: 결제 도메인 설계 및 백엔드 개발 (부팀장)
- **Key Point**: `@Transactional`을 활용한 결제 원자성 보장, 외부 API 장애 대응
- **Troubleshooting**: 네트워크 오류 시 결제 데이터 불일치 방지를 위한 서버 측 금액 검증 로직 구현
- [Repo Link](https://github.com/devHyeRin/Project_LearnIT)

### 2. Let's Gu (Authentication & Event Platform)
- **Role**: 이벤트 CRUD 개발 및 시스템 로직 분석
- **Key Point**: 로그인 인증 흐름 분석 및 세션 결함 수정
- **Troubleshooting**: 담당 영역이 아닌 로그인 로직 결함까지 분석하여 서비스 안정성 복구
- [Repo Link](https://github.com/devHyeRin/Project_LetsGu)

### 3. Fluffy Puppy (Refactoring Project)
- **Role**: 레거시 리팩토링 및 CI/CD 인프라 구축
- **Key Point**: QueryDSL 성능 최적화(N+1 개선), Docker-compose 배포 자동화
- **Troubleshooting**: 분리된 회원 인증 구조(로컬/SNS) 통합 및 HTTPS 적용을 통한 보안성 강화
- [Repo Link](https://github.com/devHyeRin/Project_FluffyPuppy)
